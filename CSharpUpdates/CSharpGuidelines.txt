
Guidelines Options
	DO NOT
	DO
	AVOID
	CONSIDER

DO use file based namespaces.
DO place global using statements into GlobalUsings.cs or into the Using element of the project file.
DO NOT access fields from outside their wrapping property.
DO declare non-nullable reference type properties as follows:
	    private string? _zip;
        public string Zip 
        { 
            get => _zip!; 
            private set => _zip=value??throw new ArgumentNullException(nameof(value)); 
        }
DO use inferred constructor when instantiating a variable of the type assigned.
	e.g. Person person = new("Inigo Montoya"); // Rather than Person person = new Person("Inigo Mongoay") 
	(and never var person = GetName("Inigo Montoya""))
DO use is null and is not null when checking null.
DO NOT specify the caller-argument-expression argument as the compiler will inject the value.
